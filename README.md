# Single-cell Atlas Zebrafish

This is a Shiny app for the Zebrafish immune single-cell atlas from:

Rovira, M. et al. A single-cell transcriptomic atlas reveals resident dendritic-like cells in the zebrafish brain parenchyma. eLife 13, (2025).


## Requirements 

The following R libraries need to be installed : Seurat, SeuratObject, DT, ggplot2, shiny, shinyjs, shinydashboard and shinydashboardPlus

## Run Shiny app

To run the Shiny app, use the following commands: 

<pre>library(shiny)
runApp()
</pre>

The Shiny App is also available online at https://scrna-analysis-zebrafish.shinyapps.io/scatlas/
